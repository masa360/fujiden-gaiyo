========================================
  事業計画書で使う画像 — 手作業での入れ方
========================================

【フォルダの場所】
  d:\cursur\slide\assets\

【やること】
  このフォルダに、次の「ファイル名」で画像を4枚入れてください。
  （中身の画像はあなたが持っている写真でOK。名前だけ合わせる。）

----------------------------------------
  1. ヒーロー用（トップの大きな写真）
----------------------------------------
  保存するファイル名（そのままコピーして使ってください）:
    hero-diorama.png

  内容の目安:
    4階建ての立体都市ジオラマ、トミカ・ミニカーが並んだ写真

----------------------------------------
  2. 「実際に何をするのか」の3枚（横並び）
----------------------------------------
  保存するファイル名:
    ① concept-fpv.png   … スマホでFPV映像を見ながらラジコンを操作している様子
    ② concept-diorama.png … トミカの世界のジオラマ都市を上から見た様子
    ③ concept-3d.png    … 3Dプリントの駐車場とカスタム車両・LEDの写真

----------------------------------------
【具体的な手順（Windowsの例）】
----------------------------------------
  1. 使いたい画像を4枚用意する（スマホやPCの写真でOK）
  2. エクスプローラーで d:\cursur\slide\assets\ を開く
  3. 画像をこのフォルダにコピー（またはドラッグ＆ドロップ）
  4. ファイル名を、上記の4つに「必ず」変更する:
       hero-diorama.png
       concept-fpv.png
       concept-diorama.png
       concept-3d.png
  5. 拡張子は .png のまま。.jpg の写真なら、コピー後に
     名前を「〇〇.png」に変えても表示されます（中身はJPEGのままでOK）

----------------------------------------
【注意】
----------------------------------------
  • ファイル名は半角英数字とハイフンだけ。スペースや日本語は入れない
  • 大文字・小文字も上記のとおりに合わせる（hero-diorama.png など）
  • 4枚とも入れたら、ブラウザで index.html を開き直す（F5で再読み込み）

これで事業計画書の画像がすべて表示されます。
